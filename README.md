# Bank Java test driven development

## Getting Started
First pull the repo :
```
git clone https://github.com/Benoit-Lorcy/bank-java-tdd
```

Then you can build the project with gradle
```
cd bank-java-tdd
.\gradlew build
```

There is no console-based staging yet 

## Libraries Used
- Lombok
- Junit
- sqlite-jdbc (sqlite driver)
